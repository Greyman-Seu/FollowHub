# PRD: 网页展示与发布

## 1. 目标

将每日 FollowHub 结果输出为高信息密度、易扫描、适合长期归档的网页资产。

## 2. 页面类型

- 每日详情页
- 最近日报索引页
- 时间归档页

## 3. 日报页结构

### 3.1 页头区

- 日期
- 汇总时间范围
- 来源概览
- 生成时间
- 失败来源提示

### 3.2 今日重点区

- 3 到 8 条重点内容
- 标题、来源、作者、简短摘要、价值说明、原文链接

### 3.3 分类摘要区

- 按内容类型或来源组织当日信息

### 3.4 论文专项区

- 论文标题
- 原文链接
- 翻译链接
- 人话解释
- 可选图像

### 3.5 全量线索区

- 高密度列表
- 标题、来源、作者、时间、原文链接

## 4. 展示原则

- 首屏给结论
- 重点先于全量
- 页面适合快速扫描
- 适合长期归档和回看

## 5. 发布规划

第一阶段先在本地生成页面。后续发布到 `https://tenstep.top/follow/`。

## 6. 非目标

- 不做第一阶段后台 CMS
- 不做第一阶段复杂交互筛选器
