# company

`company/` 用于承载 FollowHub 从产品定义到上线发布的全过程文档。

## 目录说明

- `00-overview/`: 产品总览与项目入口
- `01-prd/`: 产品需求文档
- `02-architecture/`: 技术架构与数据设计
- `03-planning/`: 实施路线图、阶段计划与验收
- `04-operations/`: 本地运行、配置、补跑与排障
- `05-release/`: 发布、上线与回滚文档
- `06-decisions/`: 关键决策记录

## 推荐阅读顺序

1. `00-overview/followhub-product-overview.md`
2. `01-prd/prd-master.md`
3. `01-prd/prd-phase1-daily-brief.md`
4. `02-architecture/system-architecture.md`
5. `03-planning/implementation-roadmap.md`
